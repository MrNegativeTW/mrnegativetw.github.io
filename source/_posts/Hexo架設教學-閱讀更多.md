# 第二種 - 無差別顯示
編輯 **主題** 的 `_congig.xml`
`enable`：啟用`true`，關閉`false`
`length`：要顯示出的字元數，也因為這樣這種方法才會叫做無差別顯示
```
auto_excerpt:
  enable: true
  length: 150
```
